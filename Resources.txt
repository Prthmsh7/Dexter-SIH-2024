# Dexter Project - External Resources

This file lists all external resources, libraries, and tools used in the development of the Dexter project. It includes links to official documentation, sources, and any relevant external references that contribute to the project's functionality and development.

## Dependencies

1. **Next.js**
   - **Description:** A React framework for building server-rendered or statically-exported React applications.
   - **Version:** 14.2.7
   - **Documentation:** [Next.js Documentation](https://nextjs.org/docs)

2. **React**
   - **Description:** A JavaScript library for building user interfaces.
   - **Version:** 18
   - **Documentation:** [React Documentation](https://reactjs.org/docs/getting-started.html)

3. **Tailwind CSS**
   - **Description:** A utility-first CSS framework for creating custom designs.
   - **Version:** 3.4.10
   - **Documentation:** [Tailwind CSS Documentation](https://tailwindcss.com/docs)

4. **Autoprefixer**
   - **Description:** A PostCSS plugin to parse CSS and add vendor prefixes automatically.
   - **Version:** 10.4.20
   - **Documentation:** [Autoprefixer Documentation](https://github.com/postcss/autoprefixer)

5. **PostCSS**
   - **Description:** A tool for transforming CSS with JavaScript plugins.
   - **Version:** 8.4.41
   - **Documentation:** [PostCSS Documentation](https://postcss.org/)

6. **ESLint**
   - **Description:** A tool for identifying and reporting on patterns in JavaScript.
   - **Version:** 8
   - **Documentation:** [ESLint Documentation](https://eslint.org/docs/user-guide/getting-started)

7. **ESLint Config Next**
   - **Description:** ESLint configuration for Next.js.
   - **Version:** 14.2.7
   - **Documentation:** [ESLint Config Next Documentation](https://nextjs.org/docs/app/building-your-application/linting)
